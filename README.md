# Opis Aplikacji:
Aplikacja mobilna umożliwiająca tworzenie bazy zdjęć przypisanych do lokalizacji.
Dzięki automatycznej akwizycji danych, każde zdjęcie zawiera współrzędne GPS.

## Sensory:
1. Aparat fotograficzny: Przechwytywanie obrazu.
2. Lokalizacja GPS: Pobieranie współrzędnych użytkownika
3. Pamięć urządzenia: Trwały zapis bazy zdjęć w pliku JSON.
4. Zegar systemowy: Automatyczne pobieranie daty i godziny wykonania fotografii.
   
### Technologie:
Jetpack Compose: Interfejs użytkownika
Navigation Compose: Przechodzenie między galerią a widokiem szczegółów.

#### Instrukcja uruchomienia:
1. Pobierz repozytorium.
2. Otwórz w Android Studio.
3. Skompiluj i uruchom na emulatorze lub telefonie (min. Android 11).
